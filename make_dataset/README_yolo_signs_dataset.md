## YOLO датасет детекции знаков (синтетика + внешние данные)

Этот модуль собирает датасет для детекции дорожных знаков в формате YOLO:
- **подмешивает внешний YOLO‑датасет** (если включено),
- **догенерирует синтетику** до целевых количеств,
- **подмешивает негативные изображения** (кадры без объектов) из папки `make_dataset/negative`,
- пишет итоговую структуру `dataset_<n>/` (картинки, YOLO‑лейблы, `dataset.yaml`, списки train/val, CSV‑аннотации, статистику).

Подробная логика и параметры находятся в коде:
- `make_dataset/config.py` — все параметры генерации
- `make_dataset/utils.py` — основная логика (шаблоны/фоны/импорт/запись/многопоточность)
- `make_dataset/effects.py` — эффекты (камера/погода/деградации)

---

## Требования

```bash
python -m pip install -r make_dataset/requirements.txt
```

---

## Входные данные (что нужно подготовить)

### 1) Описание классов
Файл: `shared/signs/signs.csv`

Колонки:
- **`filename`**: имя файла шаблона (обычно `*.svg`) или базовый “код” знака
- **`class_id`**: целочисленный id класса (используется в YOLO‑лейблах)
- **`class_name`**: имя класса (пишется в `classes.txt`/`dataset.yaml`)

### 2) Шаблоны знаков (основные и вариации)
- Основные: `shared/signs/images/`
- Вариации: `shared/signs/splits/<name>/...`

### 3) Фоны
Два источника фонов (используются вместе):
- `make_dataset/dashcam_frames` — кадры видеорегистратора (ищутся **рекурсивно**)
- `make_dataset/backgrounds` — дополнительные фоны (ищутся **не рекурсивно**)

Важно: папку `make_dataset/dashcam_frames` нужно скачать отдельно с Яндекс.Диска и распаковать в `make_dataset/`.
Ссылка: [dashcam_frames](https://disk.yandex.ru/d/VyC2czLyajINLw)

### 4) Внешний YOLO‑датасет (опционально)
Папка: `cfg.EXTERNAL_DATASET_DIR`

Ожидается структура “как у YOLO” (внутри есть `data.yaml`, `images/`, `labels/`).

### 5) Негативные изображения (кадры без знаков)
Папка: `make_dataset/negative` (или `cfg.NEGATIVE_IMAGES_DIR`)

Важно: папку `make_dataset/negative` нужно скачать отдельно с Яндекс.Диска и распаковать в `make_dataset/`.
Ссылка: [negative](https://disk.yandex.ru/d/gz2vv16C3n4BmA)

---

## Как запустить

Из корня репозитория:

```bash
python make_dataset/generate_synth_yolo_dataset.py
```

Рекомендация: сначала сделать маленький тест-прогон (уменьшить `MIN_IMAGES_PER_CLASS`, отключить внешний датасет/многопоточность),
а потом включать “боевые” параметры.

---

## Что получается на выходе

Скрипт создаёт новую папку `dataset_<n>` (точнее: `cfg.OUTPUT_BASE_<n>`), где `<n>` авто‑инкрементится.

Структура:
- `images/` — изображения
- `labels/` — YOLO‑лейблы `.txt` (или пустые файлы для негативов)
- `annotations.csv` — плоский CSV со всеми bbox
- `classes.txt` — список имён классов
- `dataset.yaml` — YAML для обучения (Ultralytics)
- `train.txt`, `val.txt` — списки путей
- `data_stats.txt` — статистика
