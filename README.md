# Дипломная работа

## Структура

```
├── README.md
├── app
├── datasets
├── make_dataset
├── models
└── training
```

* app - Встраиваемое приложение
* datasets - Директория с датасетами разных версий
* make_dataset - Директория скрипта для создания искуственного датасета
* models - Директория с обученными моделями
* training - Директория с скриптами для обучения
